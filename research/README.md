# Research Workspace

Research artifacts are separate from product docs and task work items.

## Structure

- `research/thesis/`: current hypotheses and claims to validate
- `research/experiments/`: experiment plans and outcomes
- `research/evals/`: fixed evaluation sets and metrics
- `research/datasets/`: dataset contracts and references
- `research/notes/`: working notes

## Rules

- Record experiments in files, not chat-only context.
- Tie product claims to explicit eval outputs.
