# Strategy Replication Datasets

Store canonical source packs for externally published strategies here.

## Layout

Each strategy gets one directory:

- `research/datasets/strategy-replication/<strategy-id>/`

Minimum contents:

- `source-summary.md`
- `source-code.pine` when code is available
- `source-notes.md`
- optional screenshots, CSV exports, or other capture evidence

## Rule

No local replication run is canonical unless its source pack exists here first.
