## 사용법

### 메시지 보내기
```bash
tools/agent-message.sh send <수신_에이전트ID> "<제목>" "<내용>"
```

### 받은 메시지 확인
```bash
tools/agent-message.sh list --unread   # 읽지 않은 메시지만
tools/agent-message.sh list            # 전체
```

### 메시지 읽기
```bash
tools/agent-message.sh read <msg_id>   # 읽음 처리 + 내용 출력
```

### 메시지 수 확인
```bash
tools/agent-message.sh count --unread
```

## 예시

```bash
# W-1003 에이전트 → W-1004 에이전트에게 핸드오프
tools/agent-message.sh send "serene-curran" \
  "W-1003 완료 — inbox 시스템 테스트 부탁" \
  "PR #1029 머지됨. state/file-ownership.jsonl 빈 파일 확인 필요."

# 받는 쪽
tools/agent-message.sh list --unread
tools/agent-message.sh read msg-20260504T120000-...
```

## 제약

- 시간당 5개, 일당 50개 (sender×receiver 쌍 기준)
- 시작 시 자동으로 unread count 표시 (tools/start.sh 통합)
- 종료 시 미답 메시지 있으면 경고 (tools/end.sh 통합)
- 메시지 파일: `state/agent-inbox/{agent_id}.jsonl` (git 추적)
