description: 세션 부팅 — Agent ID 발번 + 현황 한 화면 보고
---

다음 절차로 세션을 부팅하세요:

1. `bash tools/start.sh` 실행
2. 출력에서 발번된 Agent ID, main SHA, open PRs, active locks, P0 priorities 확인
3. `work/active/CURRENT.md` 읽고 활성 Work Items 표 확인
4. `memory/sessions/agents/` 디렉토리에서 직전 에이전트 jsonl의 마지막 줄(인계) 추출
5. 사용자에게 한 화면으로 정리 보고:

```
Agent: <ID>
Baseline: <main SHA 8자리> (origin/main)
Branch: <현재 브랜치>

Open PRs: <목록 또는 (none)>
Active locks: <목록 또는 (none)>

P0: <한 줄 요약>
직전 인계 (<직전 Agent ID>):
  shipped: ...
  next: ...
```

6. 보고 끝에 추천 다음 액션 1줄 (예: "/잠금 <도메인>으로 작업 시작 권장").

자유 텍스트 추가 금지 — 위 형식만.
