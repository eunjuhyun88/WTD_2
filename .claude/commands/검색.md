검색어: $ARGUMENTS

다음 위치를 모두 검색:

1. `memory/decisions/*.md` — 영구 결정 (왜)
2. `memory/incidents/*.md` — 사고 + 수정
3. `memory/questions/*.md` — 열린 질문
4. `memory/metrics/*.jsonl` — 시간 추이 수치
5. `memory/sessions/**/*.jsonl` — 이벤트 로그
6. `design/proposed/*.md` `design/current/**/*.md` — 계약/설계
7. `spec/ROADMAP.md` `spec/PRIORITIES.md` — work item 요약
8. `work/active/*.md` `work/completed/*.md` — work item 본문

가능하면 `mk.evidence_first("$ARGUMENTS")` 호출 후 결과 보강.

결과 형식 (관련도 순, 최대 10개):

```
[유형] 출처:line
  요약: <한 줄>
  날짜: <YYYY-MM-DD>
```

매칭 없으면 "기록 없음 — 새로 /결정 또는 /물음으로 만드세요" 안내.
