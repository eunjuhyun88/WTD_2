---
description: 도메인 lock — 멀티 에이전트 충돌 차단
argument-hint: <도메인 경로 또는 콤마 구분 목록>
---

도메인: $ARGUMENTS

절차:

1. `bash tools/claim.sh "$ARGUMENTS"` 실행

2. 거절되는 경우 (다른 에이전트가 같은 도메인 보유):
   - `spec/CONTRACTS.md`에서 충돌 행 추출해서 보여주기 (Agent / Domain / Branch / Started)
   - 사용자에게 3가지 선택지 제시:
     a) 기다림 (대기)
     b) 다른 도메인으로 변경
     c) 강제 잠금 (위험 — 명시적 "강제" 입력 필요)
   - "강제"를 입력해야만 강제 잠금 진행

3. 성공 시 보고:
   ```
   Locked: <도메인>
   Agent: <ID> | Branch: <현재 브랜치> | Started: <KST 시각>
   ```

4. 잠금 후 추천 다음 액션 1줄 (예: "이제 작업 시작. 종료 시 /닫기").

`/닫기`가 자동으로 lock 해제하므로 별도 풀기 명령 없음.
