description: 다음 에이전트용 인계 기록 (보통 /닫기가 자동 호출)
---

이번 세션 활동을 자동 추출 후 다음 구조를 채워서 사용자에게 초안 제시:

```
한 일:
  - <PR #번호 또는 커밋 hex>: <한 줄 요약>
  - <PR # 또는 hex>: <한 줄>
  - ...

안 한 일 (의도적으로 스코프 외):
  - <항목>
  - 또는 "없음"

막힌 곳:
  - <블로커 또는 알 수 없었던 것>
  - 또는 "없음"

즉시 다음 단계 (다음 에이전트 첫 5분):
  1. <행동 1>
  2. <행동 2>

전제 가정 (깨지면 위 다음 단계 무효):
  - main SHA: <8자리>
  - 의존 PR: #번호 (상태)
  - 환경: <중요한 환경 가정 — 예: GCP cogotchi-00013-c7n 서빙 중>
```

**필수 규칙**:
- "한 일"은 **검증 가능한 산출물** (PR/커밋/파일 경로). "이것저것 정리" 금지
- "즉시 다음 단계"는 **명령 형태로** ("X 검토" 아니라 "`bash tools/start.sh`로 부팅 후 PR #X 리뷰")
- "전제 가정"의 main SHA는 반드시 명시 — `git rev-parse --short origin/main`로 확인

자동 추출 소스:
- `git log <세션 시작 SHA>..HEAD --oneline`
- `gh pr list --search "involves:@me created:>YYYY-MM-DD"`
- 대화 맥락 (사용자 의도 / 막힘 표현)
- `state/state.json` (현재 main, open PRs)

사용자 [y/수정/취소] 확인 후 저장:
- `memory/sessions/agents/<Agent ID>.jsonl`에 append (전체 구조를 한 JSON 객체로)
- `memory/sessions/<YYYY-MM-DD>.jsonl`에도 동일 객체 append (날짜 인덱스용)

저장 후 보고: 두 파일 경로 + Agent ID.
