`tools/verify.py`에 위임합니다. 직접 실행:

```bash
./tools/verify.py $ARGUMENTS
```

**verify.py가 자동으로:**
1. 변경 파일 분석 (engine/app/docs 분류)
2. 품질 grep (보안 이슈)
3. engine tests (`uv run pytest engine/tests/ -x -q`)
4. app typecheck (`pnpm typecheck`)
5. PASS/BLOCK 결과 출력 + 실패 시 remediation

**BLOCK 시**: verify.py 출력의 "다음:" 줄을 실행해 해결 후 재실행.

docs-only 변경이면 자동으로 테스트 스킵.
`--quick` 플래그: pytest + 품질 grep만 (typecheck 생략).
