## [SECTION 1] HERO

### 카피 구조

```
EYEBROW:  COGOCHI
H1:       내 판단을 학습하는 나만의 트레이딩 AI.
SUB:      패턴을 저장하고 → 6년치 데이터로 검증하고 → 실시간 시그널로 받는다.
          쓸수록 나를 닮고, 나보다 정확해진다.
```

### Start Bar (핵심 인터랙션)

```
┌────────────────────────────────────────────────────────┐
│  What setup do you want to track?  __________________ │
│  예: btc 4h bollinger_expansion + recent_rally        │
└────────────────────────────────────────────────────────┘
           [Track This Pattern →]
```

- 인풋 필드에 타이핑하면 자동완성 드롭다운 표시 (토큰·인디케이터·블록명)
- 엔터 또는 버튼 클릭 → 입력값을 쿼리 파라미터로 담아 `/terminal?q={입력값}` 이동
- 빈 상태에서 엔터 → `/terminal` 이동 (입력 없이도 OK)
- Placeholder 텍스트는 3초마다 자동 순환:
  - "BTC 4H bollinger squeeze + OI 급증"
  - "ETH 1H RSI 과매도 + CVD 반등"
  - "SOL 숏 포지션 과열 + 펀딩비 역전"

### CTA 버튼

| 버튼 | 목적지 | 스타일 |
|------|--------|--------|
| `Open Terminal` | `/terminal` | Primary, 밝은 색 |
| `See How Lab Scores It →` | `/lab` | Secondary, 텍스트 링크 |

### 배경 비주얼

- 거의 검정 배경 (`#0a0a0a` 기준)
- COGOCHI 워터마크 로고 엠보싱 (opacity 0.04, 화면 가득)
- 우측 또는 배경에 실시간 캔들차트 미니어처 (BTC 1H, 읽기전용, 데코레이션용)
- 별도 3D 모션, 궤도 카드 없음 — 타이포그래피가 주역

## [SECTION 2] LEARNING LOOP

### 목적

"Cogochi가 어떻게 작동하는가"를 4단계로 시각화. 사용자는 각 단계를 보며 "아, 이렇게 쓰는 거구나"를 느낀다.

### 구성

```
01 CAPTURE         02 SCAN            03 JUDGE           04 DEPLOY
──────────         ───────            ───────            ────────
Terminal에서        저장된 패턴을       매칭된 자리에서     검증된 패턴만
차트를 보고         30개 코인 ×        PnL로 판정.        실시간 시그널로.
자연어로 패턴 구성.  6년치 데이터 검색.  통과 기준 6조건.   알림 수신 + 피드백.
```

### 동작 상세
- 4개 카드는 가로 배열 (데스크탑) / 세로 배열 (모바일)
- 각 카드 호버 시 카드 살짝 부각 (border glow)
- 카드 클릭 시 해당 Surface로 이동:
  - 01 Capture → `/terminal`
  - 02 Scan → `/lab`
  - 03 Judge → `/lab` (메트릭 탭)
  - 04 Deploy → `/dashboard`
- 숫자(01~04)는 모노스페이스 폰트, 크게

## [SECTION 3] 3-SURFACE 소개 카드

### 목적

Terminal / Lab / Dashboard 각 페이지가 무엇인지 한눈에.

### 카드 구성 (3개)

#### 카드 1 — TERMINAL
```
아이콘:  📡 (또는 차트 픽셀 아이콘)
제목:    Terminal
한 줄:   보고 판단한다
설명:    차트를 보고, 자연어로 패턴을 구성하고,
         한 클릭으로 Challenge에 저장.
CTA:     Open Terminal →
```

#### 카드 2 — LAB
```
아이콘:  ⚗ (또는 분석 픽셀 아이콘)
제목:    Lab
한 줄:   검증하고 개선한다
설명:    저장한 패턴을 6년치 데이터로 돌린다.
         SCORE, 승률, 기댓값이 숫자로 나온다.
CTA:     Open Lab →
```

#### 카드 3 — DASHBOARD
```
아이콘:  ⌂ (또는 인박스 픽셀 아이콘)
제목:    Dashboard
한 줄:   내 작업 인박스
설명:    내 Challenge, 워치 목록, 알림 히스토리.
         지금까지 만든 것들이 여기 있다.
CTA:     Open Dashboard →
```

### 동작 상세
- 3열 그리드 (데스크탑) / 세로 스택 (모바일)
- 카드 배경: 약간 밝은 패널 (`#111` 기준), 얇은 테두리
- CTA 클릭 → 해당 라우트 이동
