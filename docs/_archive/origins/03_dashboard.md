# COGOCHI — `/dashboard` Dashboard 페이지

> 역할: 내 작업 인박스  
> 핵심: "내가 만든 것들, 지금 돌아가고 있는 것들, 피드백을 기다리는 것들"  
> 복잡한 분석 없음. 상태 파악 + 빠른 이동이 전부

---

## [HEADER] — 대시보드 헤더

```
[◀ COGOCHI]    DASHBOARD    [⚙ 설정]    [🔔 알림 N개]
```

| 요소 | 동작 |
|------|------|
| `◀ COGOCHI` | 홈(`/`) 이동 |
| `⚙ 설정` | 설정 슬라이드오버: 알림 설정, Telegram 연결, 플랜 관리 |
| `🔔 알림 N개` | 미확인 시그널 알림 수 배지. 클릭 → Section 3 스크롤 |

## [SECTION 1] MY CHALLENGES

### 역할

Lab Challenge의 요약판. 최근 5개 표시. 전체 보기는 Lab으로 이동.

### 구성

```
MY CHALLENGES                                    [Lab 전체 보기 →]
─────────────────────────────────────────────────────────────────

┌─────────────────────────────────────────────────────────────┐
│  ✅  btc-macd-style          SCORE +0.0234      2h ago      │
│      long · 1H · 359개 인스턴스 · PASS                      │
│      [Lab에서 열기 →]                                        │
├─────────────────────────────────────────────────────────────┤
│  ❌  sample-rally            SCORE -1.000       8h ago      │
│      long · 1H · 1221개 · FAIL                              │
│      [Lab에서 열기 →]                                        │
├─────────────────────────────────────────────────────────────┤
│  ─   exhaustion-short        미실행                         │
│      short · 4H · 평가 필요                                  │
│      [평가하러 가기 →]                                       │
└─────────────────────────────────────────────────────────────┘

[+ New Challenge  →  Terminal]
```

### 동작

| 동작 | 결과 |
|------|------|
| 행 클릭 / `[Lab에서 열기]` | `/lab?challenge={slug}` 이동 |
| `[평가하러 가기]` | `/lab?challenge={slug}` + 자동 포커스 |
| `[+ New Challenge]` | `/terminal` 이동 |
| `[Lab 전체 보기]` | `/lab` 이동 |

### 상태 아이콘

| 아이콘 | 의미 |
|--------|------|
| ✅ | Stage 1 PASS |
| ❌ | Stage 1 FAIL (Graveyard) |
| `─` | 미실행 |
| `⚙` | 평가 진행 중 |
| `📡` | 시그널러 배포 중 |

## [SECTION 2] WATCHING

### 역할

Terminal에서 저장한 라이브 검색 항목. "지금 이 패턴 계속 모니터링 중"인 것들.

### 구성

```
WATCHING                                      [Terminal에서 추가 →]
─────────────────────────────────────────────────────────────────

┌─────────────────────────────────────────────────────────────┐
│  📡  BTC 4H   recent_rally + bb_expansion    ✅ live        │
│      마지막 확인: 3분 전                                      │
│      [Terminal에서 열기 →]  [⏸ 일시정지]  [🗑]             │
├─────────────────────────────────────────────────────────────┤
│  ⏸  ETH 1H   rsi_oversold + cvd_bounce      ⏸ paused      │
│      일시정지: 2일 전                                         │
│      [Terminal에서 열기 →]  [▶ 재개]  [🗑]                  │
└─────────────────────────────────────────────────────────────┘

비어있을 때:
┌─────────────────────────────────────────────────────────────┐
│  아직 워치리스트가 없어요.                                    │
│  Terminal에서 패턴 검색 후 저장해보세요.                      │
│  [Terminal 열기 →]                                          │
└─────────────────────────────────────────────────────────────┘
```

### 동작

| 동작 | 결과 |
|------|------|
| `[Terminal에서 열기]` | `/terminal?q={쿼리}` 이동 |
| `[⏸ 일시정지]` | 스캔 중지, 상태 paused 전환 |
| `[▶ 재개]` | 스캔 재개, 상태 live 전환 |
| `[🗑]` | 삭제 확인 모달 → 삭제 |
| `[Terminal에서 추가]` | `/terminal` 이동 |

### 상태

| 상태 | 의미 |
|------|------|
| ✅ live | 현재 스캔 중 |
| ⏸ paused | 일시정지 |
| ⚠ error | 스캔 에러 (데이터 오류 등) |

## [SECTION 3] SIGNAL ALERTS

### 역할

시그널러가 감지한 알림 목록. 피드백(✓/✗) 대기 중인 것 우선 표시.  
"내 패턴이 실제로 시장에서 발생했는가?"를 확인하고 피드백을 남기는 공간.

### 구성

```
SIGNAL ALERTS                          [Telegram 연결 ✅]  [설정]
─────────────────────────────────────────────────────────────────
[탭: 피드백 대기 (3) | 전체 히스토리 | 자동 판정됨]

┌─────────────────────────────────────────────────────────────┐
│  🔔 btc-macd-style  감지됨                                   │
│     BTCUSDT · 4H · LONG · Alpha 점수: 82/100               │
│     2026-04-13 14:32                                        │
│                                                             │
│     [✓ 맞아]   [✗ 아니야]   [차트 보기 →]                   │
│                                                             │
│     ─ 1시간 후 자동 판정: 대기 중 ─                          │
├─────────────────────────────────────────────────────────────┤
│  🔔 btc-macd-style  감지됨                       자동 판정됨  │
│     SOLUSDT · 4H · LONG                                    │
│     2026-04-12 09:15                                        │
│                                                             │
│     결과: ✅ HIT  (+1.3% @ 1H)                              │
│     [차트 보기 →]                                           │
└─────────────────────────────────────────────────────────────┘
```

### 알림 상세 상태

| 상태 | 의미 |
|------|------|
| `피드백 대기` | 발생한 지 1시간 이내, 수동 피드백 가능 |
| `자동 판정됨 HIT` | 1시간 후 +1% 이상 이동 |
| `자동 판정됨 MISS` | 1시간 후 -1% 이하 이동 |
| `자동 판정됨 VOID` | 1시간 후 ±1% 범위 내 |

### 피드백 동작

| 버튼 | 저장 내용 |
|------|----------|
| `[✓ 맞아]` | feedback: agree, outcome_label: good |
| `[✗ 아니야]` | feedback: disagree, outcome_label: bad |
| 피드백 후 | 해당 알림 "자동 판정됨" 탭으로 이동 |

### 탭 구성

| 탭 | 내용 |
|----|------|
| 피드백 대기 | 수동 피드백 아직 안 한 알림 (최신순) |
| 전체 히스토리 | 모든 알림 (최신순, 페이지네이션) |
| 자동 판정됨 | 1시간 후 자동 판정된 알림 모음 |

### 비어있을 때

```
아직 시그널이 없어요.
Lab에서 패턴을 평가하고 시그널러를 배포해보세요.
[Lab 열기 →]
```

## 상태 / 예외 처리

| 상황 | 처리 |
|------|------|
| Challenge 없음 | Section 1에 빈 상태 안내 + `[+ New Challenge]` |
| 워치리스트 없음 | Section 2에 빈 상태 안내 + `[Terminal 열기]` |
| 알림 없음 | Section 3에 빈 상태 안내 + `[Lab 열기]` |
| Telegram 미연결 | Section 3 상단 경고 배너: "알림을 받으려면 Telegram을 연결하세요" |
| Pro 전환 모달 | 한도 도달 시 자동 표시, 닫기 가능 |
