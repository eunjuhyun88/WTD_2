# 실행 계획: 형사고소 제출세트 변호사 수준으로 완성

바로 실행. 현은주의 실제 상황 반영:
- 2023년 4월 김기태(=김민준)에게 2억원을 주고 양평 휘페스타5차 분양계약
- 토지주 방종환의 도장사용승낙서부터 위조/무단사용
- 방종환 본인이 녹음에서 "나는 모른다" 진술 → 팔 수도 없는 땅을 팔아먹음
- 공정증서 작성했으나 16일 후 경매 → 돈 갚을 능력 없었음
- 2년 넘게 "망했다, 시간달라" 하면서 제너두그룹 대표로 활동
- "용역이다, 주단위로 쪼금씩 받는다" 거짓말하며 500만원만 줌
- 매달 돈 준다 기망하다 결국 배째라, 답장 안 함
- 자해 위협으로 협박
- 공정증서 약정이자 연 15%
- 피해자는 길바닥에 나앉을 위기

실행:
1. MCP 연결 → 법률 검증
2. 고소장 전면 재작성 (실제 상황 반영)
3. 증거제출목록 재작성
4. 피해금액산정서 (이자 15% 포함)
5. 폴더 정리 + 파일명 증거번호 통일
6. 변호사 분석 보고서
7. 최종 검증
