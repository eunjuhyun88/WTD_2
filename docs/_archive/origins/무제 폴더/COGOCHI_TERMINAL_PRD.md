## 1. 핵심 가치

> 트레이더가 "어? 이럴 때마다 이렇게 움직이네"라고 느끼는 순간,
> 그 직감을 검증할 데이터를 즉시 보여주고,
> 검증된 패턴을 저장해서 Scanner가 24시간 감시하게 만든다.

### 그 순간에 필요한 것

```
1. 가격이 어떻게 움직였나      → 차트
2. 그때 시장 내부가 어땠나     → 오더플로우 (CVD, Delta, Taker)
3. 파생상품 구조가 어땠나      → OI, FR, L/S, 청산
4. 호가창이 어땠나            → 매수/매도벽, 불균형
5. 거시적 구조가 어땠나        → HTF, 와이코프, VWAP
6. 외부 요인이 있었나          → 온체인, 섹터, 심리
7. 이게 반복되는 패턴인가      → 과거 비교
```

## 6. DOUNI 역할 (채팅에서)

### 6.1 데이터 해석

```
유저: "BTC 4H 봐줘"

DOUNI:
  1. Artifact에 차트 자동 로드 (4H 캔들 + CVD)
  2. 15레이어 계산 → SignalSnapshot
  3. 채팅에 핵심 요약:
     "CVD가 베어리시 다이버전스야. 가격은 횡보인데 매수 체결이 줄고 있어."
  4. 메트릭 카드 인라인 (OI, FR, L/S)
  5. 레이어 요약 카드
```

### 6.2 추가 분석 요청

```
유저: "OI도 봐줘"
→ Artifact 파생 탭으로 자동 전환
→ OI 추이 차트 + DOUNI 해석

유저: "청산존 어디야"
→ 청산 히트맵 로드
→ "여기 롱 $4.2M 클러스터 있어. 저기까지 내리면 캐스케이드야."

유저: "HTF 구조 봐봐"
→ Artifact 구조 탭 전환
→ BOS/CHoCH + 와이코프 표시
```

### 6.3 패턴 발견 → 저장

```
유저가 ✓ 1개 이상 누름 → [📌 패턴 저장] 활성화

DOUNI: "좋아! 이 패턴 저장하면 Scanner가 24시간 찾아줄게."

저장 모달:
  패턴명: "CVD다이버전스+펀딩과열" (자동 생성, 수정 가능)
  방향: SHORT
  조건: l11.cvd_state = BEARISH_DIVERGENCE
        l2.fr > 0.0010
        l1.phase = DISTRIBUTION

저장 → Doctrine 추가 → Scanner 즉시 활성화
```

### 6.4 과거 비교

```
유저: "이거 전에도 있었어?"

DOUNI: "이 조합이 지난 3개월에 8번 있었어.
       6번은 이후 2시간 내 -2%+ 이동했어.
       적중률 75%."

       ┌─ 과거 발생 ──────────────────────┐
       │ 3/15 BTC 4H  ✓ -3.2%  2H 후     │
       │ 3/08 BTC 4H  ✓ -2.1%  1H 후     │
       │ 2/28 ETH 4H  ✗ +0.5%  (무효)    │
       │ 2/20 BTC 4H  ✓ -4.8%  3H 후     │
       │ ...                               │
       └───────────────────────────────────┘
```

## 9. 빌드 순서

### Phase 1a (Week 3~4): 핵심 대화 + 차트

```
□ 채팅 UI (ChatGPT 스타일, 중앙 정렬)
□ Artifact 패널 (슬라이드 열림/닫힘)
□ 차트 탭 (Lightweight Charts 연동)
□ 인텐트 분류 (ANALYZE → 차트 로드)
□ DOUNI 프롬프트 → llmService SSE 스트리밍
□ 인라인 메트릭 카드 (OI, FR, L/S)
□ 인라인 레이어 요약 카드
□ 분석 스택 (✓/✗ 반응)
□ [📌 패턴 저장] → Doctrine API
```

### Phase 1b (Week 5~6): 고급 데이터

```
□ 오더플로우 탭 (CVD 차트 + Delta Bar + Taker비율)
```
