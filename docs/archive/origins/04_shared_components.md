## 글로벌 네비게이션

### 데스크탑

```
┌──────────────────────────────────────────────────────────────────┐
│  COGOCHI   BTC $84,230 +1.2%  ETH $2,340 -0.3%  SOL $142 +2.1%  │
│            ─────────────────────────────────────────────────     │
│            TERMINAL  ·  LAB  ·  DASHBOARD         [⚙] [Connect] │
└──────────────────────────────────────────────────────────────────┘
```

| 위치 | 요소 | 동작 |
|------|------|------|
| 좌측 | COGOCHI 로고 | `/` 이동 |
| 중앙 좌 | BTC/ETH/SOL 실시간 가격 | Binance WS, 변동 시 색상 플래시 |
| 중앙 우 | TERMINAL · LAB · DASHBOARD | 현재 페이지 강조(밑줄 or 굵기) |
| 우측 | `⚙ 설정` | Dashboard 설정 슬라이드오버 |
| 우측 | `Connect` | 지갑 연결 (Phase 2+, 현재 비활성) |

### 모바일

```
┌───────────────────────────────────────────────────────────────┐
│  ⌂        💬 TERMINAL       ⚗ LAB       @ DASHBOARD          │
└───────────────────────────────────────────────────────────────┘
```

- 하단 탭바 고정
- 아이콘 + 라벨 (라벨은 선택된 탭만 표시 or 전체 표시)
- 현재 탭 포인트 컬러 강조

### 활성 탭 표시

| 규칙 | 방법 |
|------|------|
| 데스크탑 | 현재 라우트 이름 밑줄 + 약간 밝은 색 |
| 모바일 | 탭 아이콘 포인트 컬러로 변경 |

## 토스트 알림 시스템

위치: 화면 우측 하단 (데스크탑) / 상단 (모바일)

| 유형 | 색상 | 예시 |
|------|------|------|
| 성공 | 초록 | "Challenge 저장 완료!" |
| 에러 | 빨강 | "저장 실패. 다시 시도해주세요." |
| 정보 | 회색 | "평가를 시작합니다..." |
| 경고 | 노랑 | "오늘 세션 2/3 사용됨" |

- 자동 사라짐: 3초 (성공/정보), 5초 (에러/경고)
- 클릭하면 즉시 닫힘
- 동시 최대 3개, 스택 표시

## 빈 상태 (Empty State) 패턴

```
[아이콘 또는 픽셀아트 부엉이]

[설명 텍스트]
아직 [아이템]이 없어요.

[주요 CTA 버튼]
```

각 섹션별 빈 상태:
- Challenges 없음 → "아직 패턴이 없어요." + `[Terminal에서 만들기]`
- Watching 없음 → "워치리스트가 비었어요." + `[Terminal 열기]`
- Alerts 없음 → "시그널이 없어요." + `[Lab에서 시그널러 배포]`

## 접근성 (Accessibility)

| 항목 | 규칙 |
