## What's proven vs not

**Proven (architecture)**:
- Framework can express any rule-based phase pattern
- Search axes (tf, duration) produce orthogonal informational surfaces
- Gate catches winners that scalar search score promotes incorrectly
- Block calibration is anchored in published research, not magic numbers
- Parameter sensitivity has plateau (not knife-edge) on tuned thresholds

**Proven (single case)**:
- TRADOOR detection produces tradeable economic signal (+36% to +111% in 48h)
- Three independent variants converge on same entry — robustness within the symbol

**Not proven (generalization)**:
- Pattern definition does not generalize across symbols
- Promotion gate measures pattern completion, not trading edge
- No forward-walk, no slippage, no realistic entry price
