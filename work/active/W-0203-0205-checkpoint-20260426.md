### W-0204 — Active Variant → Live Monitor 자동 연동

**파일**: `engine/api/routes/captures.py`

**변경**: `POST /{capture_id}/benchmark_search` 라우트에서
`run_pattern_benchmark_search()` 호출 시 `active_variant_store=None` → `active_variant_store=ACTIVE_PATTERN_VARIANT_STORE` 로 수정.

**효과**: benchmark search에서 gate를 통과한 winner variant가
`ACTIVE_PATTERN_VARIANT_STORE`에 자동 upsert. 이후
`scan_all_patterns_live()` / `search_pattern_state_similarity()`
가 다음 호출 시 신규 variant를 자동 반영.

이전에는 `run_pattern_benchmark_search()` 내부에서 `active_variant_store is not None` 조건으로 분기하지만 라우트에서 `None`으로 넘겨서 레지스트리 업데이트가 항상 skip됐음.
